# Tic-Tac-Toe-game-in-python
A simple and interactive Tic Tac Toe game built in Python. This project allows two players to play Tic Tac Toe in a clean and easy-to-use console interface. It includes turn-based gameplay, win/draw detection, and automatic board updates after each move. Perfect for beginners learning Python, functions, and basic game logic.

# A fully functional Tic Tac Toe game developed using Python, featuring a clean console interface and smooth turn-based gameplay.

## The project includes:

Dynamic 3×3 game board

Real-time board updates

Win, loss, and draw detection logic

Input validation for player moves

Easy-to-read and beginner-friendly code

This project is ideal for Python learners who want to practice conditionals, loops, lists, and game logic implementation.

# Features

✔ GUI made using Tkinter
✔ Two-player mode
✔ Green highlight on winning tiles
✔ Message box showing winner
✔ Turn indicator label
✔ Clean, beginner-friendly Python logic

# How It Works

Player X starts the game

Players click boxes to mark X or O

After every move, winning combinations are checked

If a winner is found, tiles turn green and a popup appears

Game ends automatically
